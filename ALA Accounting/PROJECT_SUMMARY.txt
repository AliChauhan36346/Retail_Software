## ?? CALCULATION FORMULAS

### Account Balance
```
Closing Balance = Opening Debit - Opening Credit + 
                  Year Debits - Year Credits
```

### Current Ratio
```
Current Ratio = Current Assets ÷ Current Liabilities
Healthy Range: 1.5 - 3.0
```

### Balance Sheet Equation
```
Assets = Liabilities + Equity
Check: |Difference| < 0.01 (accounting precision)
```
